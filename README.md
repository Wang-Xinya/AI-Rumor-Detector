# 2026《人工智能导论》大作业 - 可解释的谣言检测

## 1.目标
基于谣言检测数据集，构建一个检测模型。该模型可以对数据集中的推文进行谣言检测，同时输出判断依据。要求： 
- 输出1：检测的输出是2分类（0代表非谣言、1代表谣言），分类准确率越高越好； 
- 输出2：判断依据的输出是一段文字，解释判断的依据； 
- 模型具有一定的泛化能力； 
- 有合理的运行时间

## 2.模型构建要求 
- 可以使用各种深度学习模型，大语言模型，RAG等技术，构造一个复合的模型。例如：用深度学习模型给出检测，大语言模型给出判断依据。
- 如果使用大语言模型开发，限定使用我校提供的模型接口，方便复现。我校提供的模型参见：https://claw.sjtu.edu.cn/guide/sjtu-api/ 
- 模型需要实现的基本功能：输入一个文本，输出一个检测和判断依据
